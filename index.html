How to use?
x

Use the M+ button to store an operation in memory.

Use the M- button to return a memory operation to the result field.

Use the M button to empty the memory.

Use the SUP button to delete characters you typed, one at a time.

Use the C button to remove an operation and result from the fields.

Use the ¬ button to type a negative number (only for negative numbers, not for subtraction).

Use the +, -, *, / buttons for basic operations, binary only. Example: 2+3, 6*7, 5/3 or 4-1.

Use the . (dot) button to enter decimal numbers.

Use the = button to calculate the operation.
